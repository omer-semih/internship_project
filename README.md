Comparative Analysis of Performance and Saturation Forecasting Capabilities in Time Series Databases: Prometheus vs. VictoriaMetrics

This repository contains the work I did during my internship. The goal of the project is to compare Prometheus and VictoriaMetrics in terms of performance and their ability to forecast saturation, meaning how well each system can help predict when resources (such as disk, memory, or query load) are approaching their limits. 

Both tools are widely used for time series data and monitoring, but they differ in architecture and scalability, so I wanted to look closely at how these differences affect real-world performance and forecasting accuracy.
